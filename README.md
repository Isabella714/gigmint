# GiGMint

## Introduction

GiGMint is a web application that syncs data from blockchain and provide complex query ability.

## Development

### Environment

- Golang 1.22.6
- MySQL

### Framework

- fiber
- gorm

### Run

- prepare `config.toml` file

- compile
```bash
go build .
```

- run server
```bash
./gigmint
```
